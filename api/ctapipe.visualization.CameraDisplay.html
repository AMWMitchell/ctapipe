CameraDisplay¶

class ctapipe.visualization.CameraDisplay(geometry, axes=None, title='Camera')[source]¶

Bases: object

Camera Display using MatPlotLib

Parameters:

geometry : CameraGeometry

Definition of the Camera/Image

axis : matplotlib.axes._subplots.AxesSubplot

a MatPlotLib Axis object to plot on, or None to create a new one

Notes

Implementation detail: Pixels are rendered as a matplotlib.collections.RegularPolyCollection, which is the most efficient way in MatPlotLib to display complex pixel shapes.
