ctapipe.io.obsconfig Module¶

Definition of Observation Configurations.

The ObsConfig class defines a hierarchy of classes that contain the configuration information related to an observation. Each sub-config should additionally be loadable on its own

This may be extended later to make a SimObsConfig that also includes extra Monte-Carlo information

When possible, items inside each class should be loaded dynamically (e.g. on first access)
