Getting StartedÂ¶

If you want to access SimTelArrray data files (recommended), you must first install the pyhessio package. This can be done by:

git clone https://github.com/cta-observatory/pyhessio
conda build pyhessio
conda install --use-local pyhessio

Next, check out the ctapipe repo:

git clone https://github.com/cta-observatory/ctapipe.git   # if you like HTTPS

cd ctapipe
make init     # will fetch required sub-repos and set up package
make develop  # will make symlinks in your python library dir

Make sure the tests and examples code finds the test and example files. Run the tests to make sure everything is OK:

make test

Build the HTML docs locally and open them in your web browser:

make doc-show

Run the example Python scripts:

cd examples
python xxx_example.py

Run the command line tools:

python setup.py install
ctapipe-info --tools

Start hacking and contributing: The following command will put the package in “developer mode”, meaning that it will make a symlink of your checked-out working directory in your local (user) python package library directory (which is usually something like $HOME/.local/lib/python/site-packages/. Then you can access the development ctapipe from anywhere on your system.

make develop
edit .

To update to the latest development version (merging in remote changes to your local working copy):

git pull

For further information, see http://astropy.readthedocs.org/en/latest/ ... most things in ctapipe work the same.
